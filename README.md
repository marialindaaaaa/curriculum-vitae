# curriculum-vitae
___Este código define uma pagina com informações pessoais, um currículo com apresentações pessoais,
ele utiliza HTML para estruturar o conteúdo, com elementos como <div>, <h1>, e <p> para organizar e exibir as informações de forma clara, ja o style.css define o estilo dele.___

___-Requere___
O <!DOCTYPE html> informa ao navegador que este é um documento HTML5. Os elementos <html>, <head> e <body> estruturam o documento.
<meta charset="utf-8">: Define a codificação de caracteres.
<meta name="viewport" content="width=device-width">: Torna o layout responsivo em dispositivos móveis.
<title>replit</title>: Define o título da página que aparece na aba do navegador.
<link href="style.css" rel="stylesheet" type="text/css" />: Inclui um arquivo CSS para estilização.
O <div id="barbie"> contém informações pessoais e uma imagem.
O conteúdo é organizado em seções usando títulos (<h1>) e separadores (<hr class="sep">), que facilitam a leitura.
Contém dados como nome, idade, endereço e número de telefone.
Descreve o meu objetivo, formação e cursos complementares.
___-Execução___
Foi estruturado para apresentar informações pessoais e acadêmicas de forma clara e organizada. Ao abrir o arquivo, o usuário verá uma foto, dados pessoais, objetivos, formação e referências. O uso de tags, como <h1> e <p>, garante uma boa hierarquia de informações, facilitando a leitura e a acessibilidade.
___-Contribuição___
As seções são separadas por cabeçalhos e linhas horizontais, facilitando a leitura.
O uso do atributo `alt` na imagem melhora a compreensão para usuários com deficiência visual.
Transmite profissionalismo e facilita a análise por recrutadores.
A inclusão do `<meta name="viewport">`
